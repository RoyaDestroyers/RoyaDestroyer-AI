# Download Instructions

La descarga se hace manualmente y luego los scripts procesan los datos localmente.

## Destinos esperados

- `data/raw/jmuben/`
- `data/raw/jmuben2/`
- `data/raw/clr_eafit/`
- `data/raw/bracol/`
- `data/raw/rocole/`
- `data/raw/uganda/`
- `data/raw/rust_miner_brazil/`
- `data/raw/roboflow_optional/`
- `data/raw/own_invalid/`

## Orden recomendado

1. JMuBEN / JMuBEN2
2. CLR EAFIT
3. BRACOL
4. RoCoLe
5. Uganda
6. Invalid images propias

## Regla

No mezclar archivos entre datasets. Cada dataset debe quedar descomprimido dentro de su carpeta correspondiente.
